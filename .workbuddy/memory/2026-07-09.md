# 2026-07-09 工作日志

## 新增：信纸风分享卡片功能
文章页选中文字 → 弹出「生成卡片」按钮 → 纯前端 Canvas 生成信纸风卡片图（霞鹜文楷 + 羊皮纸横纹 + 署名 + 文章链接二维码）→ 弹层预览 → 下载/长按保存，用于朋友圈/小红书分享。

### 文件变更
- 新增 `themes/letter/source/js/card.js`：核心逻辑（选区监听、字体预加载、Canvas 绘制、二维码、下载），IIFE 风格
- 新增 `themes/letter/source/css/card.css`：浮动按钮 + 弹层样式，复用 style.css CSS 变量
- 新增 `source/vendor/qrcode-generator.min.js`：二维码库（本地，纯 JS 无依赖，UMD 全局名 `qrcode`）
- 修改 `themes/letter/layout/post.ejs`：文末注入 `#card-meta`（data-title/date/url/site）+ 引入 card.css/qrcode/card.js（仅文章页加载）

### 关键实现点
- **字体**：Canvas 用 `'LXGW WenKai'`（与 `source/fonts/lxgwwenkai-regular.css` 的 @font-face family 一致；style.css 里 TsangerJinKai 是失效声明）。绘制前 `document.fonts.load('48px "LXGW WenKai"', text)` 触发对应 unicode-range 子集加载，`document.fonts.ready` 兜底。
- **画布**：1080×1350（4:5）@DPR=2 实际 2160×2700，适配小红书/朋友圈。
- **信纸复刻**：羊皮纸底 #f5f4ed + 卡片 #fbfaf4 + 横纹（间距 = fontSize×1.85，复刻 1.85em）+ 左侧墨蓝竖线（呼应 blockquote）+ 40% 宽分隔线（呼应 hr）。
- **文字**：逐字 measureText 换行 + 行首标点禁则；超长自适应缩字号（50→30）再截断 12 行加省略号。
- **二维码**：qrcode(0,'M') 生成矩阵手画 canvas，模块色墨蓝 #1B365D，失败降级文字链接。
- **交互**：mouseup/touchend 监听 .post-content 选区，浮动按钮 fixed 定位（避让边界）；移动端 iOS 兜底底部入口 + textarea 编辑。
- **下载**：canvas.toBlob + a.download，文件名 `写给你的信-标题-日期.png`；移动端长按图片保存。
- **元信息**：post.ejs 用 `config.url + url_for(page.path)` 拼完整 URL，中文被 encodeURI（二维码扫码安全）。
